Sliding Puzzle Game,
Advanced Programming Project - L2

As part of the advanced programming course, we had to implement a sliding puzzle game in pairs and within a week. With my classmate Thomas Ekindy, we stood out with a particularly aesthetic desktop application. With a strong visual identity, a selection of 54 randomly played music tracks, and a save and scoring system, we managed to turn an academic exercise into a truly fun little product.
I warmly invite you to try our game by retrieving the files from the GitHub repository following the instructions in the readme.

Technologies Used

  • C, SDL library
  • Makefile, automated compilation
  • Latex (report writing)

Transversal Skills

  • Teamwork, task allocation
  • Report writing in Latex
