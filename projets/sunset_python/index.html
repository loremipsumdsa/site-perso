Sunset - Module Python pour l'embarqué,
Projet étudiant, 2023

Dans le cadre de l'enseignement linux embarqué, nous devions réaliser un projet sur Raspberry PI. Avec mon camarade Thomas, nous avons développé un package python pour le TCN75A, un capteur de température que nous avions déjà utilisé précédement (voir projet sunset).
Le module s'appuie sur smbus . Il permet de communiquer avec le capteur pour le configurer, lire la température, le mettre ou le sortir de veille ou encore effectuer une capture one-shot.
Un effort particulier à été fait sur la propreté du code et son auto-documentation. Enfin pour tester notre module nous avons développé une petite interface web qui permet d'intéragir avec le capteur à distance via le réseau local.

Technologies utilisées

  • Python
  • Raspberry PI 3 B
  • Flask
